# Paper 1 Figures

Planned figures for the first short paper draft:

1. `wilsonian_hierarchy.pdf`
   - schematic of the $2\times2$ split into one coarse scalar and three fluctuation modes
   - show recursive propagation of the coarse field and local trivialization of the fluctuation field

2. `std_deltaS_scaling.pdf`
   - volume scaling of `std(ΔS)` and `std(ΔS)/L`
   - include current `16^2` and `32^2` points, and later `64^2` if available

3. `knockout_importance.pdf`
   - bar chart of the HMC knockout ranking by increase in `std(ΔS)`

4. `conditional_decomposition.pdf`
   - per-level conditional negative log-probability contribution, either total or per component

5. `rw_histograms.pdf`
   - optional comparison of reweighting-factor histograms between the old baseline and the Wilsonian transport-map model

The manuscript uses placeholders if these files are missing, so the paper still compiles while figures are being prepared.
