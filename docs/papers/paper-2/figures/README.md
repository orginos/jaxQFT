# Paper 2 Figures

Planned figure set for the RG/universality paper:

1. `blocked_sampler_hierarchy.pdf`
   - exact blocked-sampler interpretation of the learned hierarchy
   - show fine model and all blocked marginals

2. `std_deltaS_intensive_scaling.pdf`
   - scaling of `std(ΔS)` and `std(ΔS)/L`
   - include `16^2`, `32^2`, `64^2`, and `128^2` if available

3. `rg_trajectory_observable_space.pdf`
   - blocked observable trajectories for several target points
   - this is the main conceptual figure

4. `blocked_observables_vs_depth.pdf`
   - `xi/L`, Binder cumulant, susceptibility, and low-momentum observables vs RG depth

5. `coupling_projection.pdf`
   - optional projected flow in a truncated coupling space

The manuscript compiles without these figures by using placeholders.
